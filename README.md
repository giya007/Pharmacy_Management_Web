# Pharmacy Management System

A simple Pharmacy Management System built using **Python Flask**, **MySQL**, **HTML**, and **CSS**.
This project was developed to practice backend development concepts such as CRUD operations, database connectivity, SQL queries, and report generation.

---

## Features

### Medicine Module

* Add medicines
* View inventory
* Update medicine details
* Delete medicines
* Search/display medicine by ID

### Customer Module

* Add customers
* View customer records
* Update customer details
* Delete customers

### Purchase History Module

* Store medicine purchase records
* Reduce stock automatically after purchase
* View purchase history

### Sales Report

* Generate sales reports by date
* Calculate total sales value

### Billing System

* Generate customer bills
* Display purchased medicines
* Calculate total bill amount

## How to Run the Project

### 1. Clone Repository

```bash
git clone <your-github-link>
```

### 2. Install Flask

```bash
pip install flask
```

### 3. Configure MySQL Database

Create the required database and tables in MySQL.

### 4. Run Application

```bash
python app.py
```

### 5. Open in Browser
